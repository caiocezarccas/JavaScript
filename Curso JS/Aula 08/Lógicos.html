Operadores Lógicos

! Negação && Conjunção "e lógico" || Disjunção "ou lógico"

Negação

Operador unário, só tem um operando

! true, false

Conjunção

Operadore binário, tem dois valores lógicos um de cada lado que vai dar resultado lógico

true && true -> true true && false -> False false && true -> False false && false -> False

Disjunção

Também é um operador binário que tem dois valores lógicos

true || true -> true true || false -> true false || true -> true false || false -> false

Precedência

() ** / - Aritméticos(tem ordem de Precedência) > < >= - Relacionais( não tem ordem de Precedência, quem aparecer primeiro faz) ! && || - Lógicos(Esse tem ordem de Precedência, e a ordem é essa sequência)

Ternário

? :

Ternário porque tem 3 partes, ele junta 3 operando

teste ? true : false media >= 7.0 ? "Aprovado" : "Reprovado"
