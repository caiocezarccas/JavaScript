Parte 2 do Operadores

Relacionais, Lógicos e Ternários

Relacionais

> Maior < Menor >= Maior ou igual <= Menor ou igual == Igual != Diferente

Exemplo: 5 > 2 True 7 < 4 False 8 >= 8 True 9 <= 7 False 5 == 5 True 4 != 4 False

Identidade

5 == 5 Tru 5 == '5' True(sinal de igualdade não testa se é String, testa a grandeza) 5 === '5' False(tem o mesmo valor, mas não tem o mesmo tipo) 5 === '5' True(tem o mesmo valor e são do mesmo tipo)
